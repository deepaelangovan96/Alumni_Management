Alumni Management System

Project Overview

The Alumni Management System is a web-based application developed to help educational institutions manage alumni information efficiently. It allows alumni to register, log in, and update their profiles while providing administrators with an organized way to manage alumni records.

Features

- Alumni Registration
- Secure Login System
- Alumni Dashboard
- View Alumni Details
- Search Alumni Records
- Update Alumni Information
- Delete Alumni Records
- User-Friendly Interface

Technologies Used

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- SQLite
- Bootstrap

Project Structure

Alumni_Management/
│── app.py
│── alumni.db
│── templates/
│── static/
│── README.md

Installation

1. Clone the repository.
2. Open the project folder in Visual Studio Code.
3. Install Flask:
   pip install flask
4. Run the application:
   python app.py
5. Open your browser and visit:
   http://127.0.0.1:5000
